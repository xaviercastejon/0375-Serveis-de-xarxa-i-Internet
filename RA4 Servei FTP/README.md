Resultat d'aprenentatge RA4 Servei FTP

L’objectiu d’aquest exercici és conèixer i implementar un servidor FTP en Ubuntu Server i verificació en un client Ubuntu Desktop. Configurarem aquest servidor per connexions SFTP i la seva administració amb Webmin.
Aquesta activitat es realitzarà en grups de 2 alumnes i es corregirà a classe en CHECK i serà part de l’avaluació del resultat d’aprenentatge de la RA4 del mòdul de Serveis de xarxa i internet.


Per realitzar aquesta pràctica necessiteu fer servir 2 màquines virtuals que seran proporcionades en l’entorn d’ISARDVDI:
- Ubuntu Server amb els serveis ja creats en la pràctica anterior:
- Servidor DNS Master amb la zona directa del domini grupX.itb.cat i
      - Servidor Web NGINX
      - Servidor proFTPD
- Client Ubuntu Desktop (anomenat grupXubuntu) per la verificació de la implementació del servidor proFTPD
