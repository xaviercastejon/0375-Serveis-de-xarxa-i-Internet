Resultats d'aprenentatge RA7 i RA8 de Serveis d'audio i de video

L'objectiu d'aquestes activitats és instal·lar i configurar:

  -  el servidor de streaming d'àudio IceCast2 amb l'eina yt-dlp, que permet descarregar vídeos i contingut multimèdia de diferents llocs web per crear una estació de ràdio a internet, així com utilitzar-los dels clients.
  - el servidor de streaming de vídeo IceCast2 amb l'eina yt-dlp, que permet descarregar vídeos i contingut multimèdia de diferents llocs web, així com utilitzar-los dels clients.



Aquestes activitats es realitzaran en grups de 2 alumnes i es corregirà a classe en CHECK i serà part de l’avaluació del resultat d’aprenentatge del RA7 i RA8 del mòdul 0375 Serveis de xarxa i Internet.


Per realitzar aquestes pràctiques necessiteu fer servir 2 màquines virtuals que seran proporcionades en l’entorn d’ISARDVDI:
- Ubuntu Server amb els serveis de:
      - Servidor DNS Master amb la zona directa del domini grupX.itb.cat i 
      - Servidor Web NGINX
- Client Ubuntu Desktop (anomenat grupXubuntu) per la verificació de la implementació del servidor de missatgeria instantània
Recordar de crear el registre DNS referent al servidor d’àudio (audio.grupX.itb.cat) i de video.
